# 🚀 Kubernetes DevOps Portfolio Project

This project demonstrates a complete DevOps CI/CD pipeline using:

- Kubernetes (Minikube)
- Docker
- Jenkins
- GitHub
- Ingress Controller

## Architecture

GitHub → Jenkins → Docker → Kubernetes → Ingress → Browser

## Services

- Frontend: Nginx app
- Backend: Node.js API
- CI/CD: Jenkins pipeline

## Features

- Kubernetes Deployments
- Services (ClusterIP + NodePort)
- Ingress routing (/ and /api)
- Docker image builds
- CI/CD pipeline (Jenkins)
